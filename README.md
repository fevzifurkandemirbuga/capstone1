## Capstone 1: Accounting Ledger Application

track all financial
transactions for a business or for personal use.

### Features
- Home Screen
  - Ledger Menu
    - Reports Menu

  ### Home Screen Menu
  - Add Deposit `addTransaction(true);`
  - Make Payment `addTransaction(false);`
  - Display Ledger `ledger()`

  ### Ledger Screen Menu
  - Display All Entries
  - Deposits
  - Payments
  - Reports `reports()`
  
  ### Reports Screen Menu
  
  - 1)Month To Date
  - 2)Previous Month
  - 3)Year To Date
  - 4)Previous Year
  - 5)Search by vendor
  - 6)Custom Search
